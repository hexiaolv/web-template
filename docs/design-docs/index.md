## 写作规范

每篇设计文档应包含：

1. **背景**：问题是什么，为什么需要决策
2. **方案对比**：考虑了哪些选项，各自的 tradeoff
3. **决策**：最终选择了什么，理由是什么
4. **后果**：这个决策带来了什么约束或影响

> 不记录"什么"，记录"为什么"——代码本身已经说明了"什么"。
