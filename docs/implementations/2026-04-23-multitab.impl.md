# 实施路径：多页签导航系统 (MultiTab)

## 1. 核心代码位置
- `src/models/multiTab.ts`：状态逻辑中心。
- `src/components/MultiTab/`：视图组件库。
- `src/app.tsx`：运行时配置入口。

## 2. 集成步骤
### Step 1: 注册全局 Model
创建 `src/models/multiTab.ts`，定义 `openTab`、`closeTab`、`refreshTab` 等原子操作。

### Step 2: 注入运行时布局
修改 `src/app.tsx` 中的 `layout` 配置：
1.  **menuItemRender**：拦截菜单点击，改为调用 `openTab`。
2.  **childrenRender**：注入 `RouteListener`、`TabBar` 及 `KeepAliveWrapper`。

### Step 3: 样式与层级适配
在 `TabBar.less` 中处理 `sticky` 定位与 `z-index: 9`。

## 3. 关键逻辑解析
### 强制刷新实现
通过更新 `reloadKey` 改变容器的 `key`，触发 React 的销毁重挂载。

### 国际化解析
在 `RouteListener` 中通过 `useIntl` 构建 `menu.admin.sub-page` 形式的 key 来还原标题。
